线性代数2 矩阵

矩阵 (Matrix)

Created at 2022-01-20 22:16:21
Updated at 2023-10-30

请阅读这篇文章作为前置知识.

矩阵是矩形的数表, 横为行, 竖为列, 对于一个 nnn 行 mmm 列的矩阵, 我们表示为 n×mn \times mn×m 矩阵, 读作 “n by m matrix”. 特别地, n×nn \times nn×n 的矩阵被称为 nnn 阶方阵.

矩阵 AAA 的第 iii 行第 jjj 列的元素可以表示为 Ai,jA_{i, j}Ai,j​. 对于方阵 AAA, 所有形如 Ai,iA_{i, i}Ai,i​ 的元素构成了 AAA 的主对角线 (main diagonal).

根据前面我们介绍的向量的定义, 我们发现矩阵本身也是向量的一种, 具有相加和数乘运算且结果仍在同一个向量空间内. 对于仅一行的矩阵, 称为行向量, 仅一列的矩阵, 称为列向量.

定义矩阵的子矩阵为这个矩阵去掉一些行或列得到的矩阵.

矩阵乘法

相对于向量本身具有的线性运算, 矩阵还有一个重要运算: 矩阵乘法.

对于 m×nm \times nm×n 矩阵 AAA, n×pn \times pn×p 矩阵 BBB, 定义矩阵乘法运算为:

[AB]i,j=∑r=1nAi,rBr,j[AB]_{i,j} = \sum_{r = 1}^n A_{i,r}B_{r,j} [AB]i,j​=r=1∑n​Ai,r​Br,j​

ABABAB 是一个 mmm 行 ppp 列的矩阵, 矩阵乘法不满足交换律, 满足分配律和结合律.

矩阵乘法的单位矩阵 InI_nIn​ 是一个 nnn 阶方阵, 主对角线全为 111, 其余部分为 000. 满足对于任意有 nnn 列的矩阵 AAA, AI=AAI = AAI=A, 对于任意有 nnn 行的矩阵 BBB, IB=BIB = BIB=B.

矩阵与线性映射

我们之前提到线性映射是函数, 将一个向量空间内的向量映射到另一个向量空间中. 结合矩阵乘法, 我们发现矩阵恰恰是一种线性映射.

当我们用一个 1×n1 \times n1×n 的行向量 AAA 乘一个 n×mn \times mn×m 的矩阵 BBB 时, 我们可以得到一个 1×m1 \times m1×m 的行向量 CCC.

Ci=∑j=1nAjBj,iC_i = \sum_{j = 1}^n A_jB_{j, i} Ci​=j=1∑n​Aj​Bj,i​

因此矩阵 BBB 就是线性映射 f(A)=A×B=Cf(A) = A \times B = Cf(A)=A×B=C.

结合矩阵也是向量的特征, 我们推广到一般情况, 线性映射也是向量, 具备线性运算, 可以构成向量空间.

逆矩阵

一个矩阵的乘法逆元是它的逆矩阵 (Inverse Matrix), 只有方阵存在逆矩阵, 定义为 A−1A=IA^{-1}A = IA−1A=I.

转置

对于 n×mn \times mn×m 矩阵 AAA, 它的转置 (transposition) 矩阵 ATA^{T}AT 是一个 m×nm \times nm×n 矩阵, 定义为:

(AT)i,j=Aj,i(A^T)_{i, j} = A_{j, i} (AT)i,j​=Aj,i​

容易理解, 矩阵转置对于矩阵的线性运算具有结合律和分配律.

秩

矩阵的秩 (Rank) 定义为最多找出多少行使得它们两两线性无关. 一个矩阵的秩等于它转置矩阵的秩. 把一个矩阵高斯消元后不全为 000 的行的数量就是这个矩阵的秩.
