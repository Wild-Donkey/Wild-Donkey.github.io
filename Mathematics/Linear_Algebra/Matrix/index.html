Matrix Multiplication

对于 mmm 行 nnn 列的矩阵 AAA, nnn 行 ooo 列的矩阵 BBB, 定义矩阵乘法运算为:

[AB]i,j=∑k=1nAi,kBk,j[AB]_{i,j} = \sum_{k = 1}^n A_{i,k}B_{k,j} [AB]i,j​=k=1∑n​Ai,k​Bk,j​

ABABAB 是一个 mmm 行 ooo 列的矩阵, 矩阵乘法不满足交换律, 满足分配律和结合律.

矩阵乘法的单位矩阵 InI_nIn​ 是一个 n∗nn * nn∗n 的矩阵, 其左上右下对角线为 111, 其余部分为 000. 满足对于任意有 nnn 列的矩阵 AAA, AI=AAI = AAI=A, 对于任意有 nnn 行的矩阵 BBB, IB=BIB = BIB=B.

Rank

矩阵的秩 (Rank) 定义为最多找出多少行使得它们两两线性无关. 一个矩阵的秩等于它转置矩阵的秩. 把一个矩阵高斯消元后不全为 000 的行的数量就是这个矩阵的秩.
