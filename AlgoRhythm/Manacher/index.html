Manacher Algorithm

“A new linear-time ‘on-line’ algorithm for finding the smallest initial palindrome of a string”
----Gleen K. Manacher (1975)

一种关于一个字符串 SSS 的回文子串的算法, 可以线性求两个数组 f1f_1f1​ 和 f2f_2f2​, 分别表示以第 iii 位为中心的最长奇回文子串和最长偶回文子串的半径 (长度除以 222, 向上取整). 对于以 iii 为中心的回文子串 [l,r][l, r][l,r], 规定 i=⌈l+r2⌉i = \lceil \frac{l + r}2 \rceili=⌈2l+r​⌉.

根据回文串的性质可以知道 f1i{f_1}_if1​i​ 和 f2i{f_2}_if2​i​ 也恰好是以 iii 为中心的回文子串的个数. 这条性质非常重要.

朴素

枚举每个回文中心 iii 每次从半径为 111 开始, 每次验证是否可以增加半径.

容易看出, 这样最坏的复杂度可达 O(n2)O(n^2)O(n2), 在 sss 形如 aaaa...aaaa 时可以出现.

不妨只考虑 f1f_1f1​. 对于 f2f_2f2​ ,除了有细节上的差别, 算法思想相同, 不再赘述.

优化

仍然是从小到大枚举回文中心 iii, 可以认为求 f1i{f_1}_if1​i​ 时, f1j (j∈[1,i)){f_1}_j~(j \in [1, i))f1​j​ (j∈[1,i)) 已知.

设 frfrfr (fr) 为 [1,i)[1, i)[1,i) 中, 令 j+f1j−1j + {f_1}_j - 1j+f1​j​−1 最大的整数. 即 [1,i)[1, i)[1,i) 中所有回文中心的最长的奇数长度回文子串中, frfrfr 的最长奇数长度回文子串右边界最靠右.

则对 iii 有两种情况:

  • i>fr+f1fr−1i > fr + {f_1}_{fr} - 1i>fr+f1​fr​−1

这时的 iii 不在任何已知的回文子串中, 则直接朴素求 f1i{f_1}_if1​i​.

  • i≤fr+f1fr−1i \leq fr + {f_1}_{fr} - 1i≤fr+f1​fr​−1

这时的 iii 在以 frfrfr 为中心的回文串内, 记 iii 关于对称轴 frfrfr 的对称点为 rerere (re). 易知 re+i=2frre + i = 2frre+i=2fr, 因此 re=2fr−ire = 2fr - ire=2fr−i.

因为 rerere 在 frfrfr 左边, 所以 f1re{f_1}_{re}f1​re​ 已知, 所以可以求出 rerere 为中心的奇数长度回文子串中, 完全被 frfrfr 的最长的奇数长度回文子串包含的最长回文子串半径为 rrr, 则这个子串关于 frfrfr 对称的子串也是回文串, 这个子串的中心恰为 rerere 关于 frfrfr 的对称点 iii.

我们已经得到了 iii 为中心的半径为 rrr 的奇数长度回文子串, 因此 f1i≥r{f_1}_i \geq rf1​i​≥r.

我们接下来讨论被 frfrfr 为中心的奇数长度包含的最长的以 rerere 为中心的奇数长度回文子串是否触及了边界:

  • i+r−1<fr+f1fr−1i + r - 1 < fr + {f_1}_{fr} - 1i+r−1<fr+f1​fr​−1

以 rerere 为中心的最长奇数长度回文子串和它外围的字符都被 frfrfr 的回文子串包含, 所以 rerere 为中心的子串不再是回文的失配位置的对称点也应当是 iii 为中心的子串不再是回文的失配位置, 因此可以确定 f1i=r{f_1}_i = rf1​i​=r.

  • i+r−1=fr+f1fr−1i + r - 1 = fr + {f_1}_{fr} - 1i+r−1=fr+f1​fr​−1

因为前面任何 f1f1f1 值都没有描述 i+r−1i + r - 1i+r−1 右边的情况, 因此在 f1i≥r{f_1}_i \geq rf1​i​≥r 的基础上继续朴素即可, 无需从 111 开始.

时间复杂度

求 Manacher 复杂度的核心在于求每次朴素的长度之和, 我们发现每次朴素地使某个核心的半径增加 111, fr+f1fr−1fr + {f_1}_{fr} - 1fr+f1​fr​−1 就会向右移动一个字符, 而根据定义这个边界永远不会往回移动. 因此朴素的次数之和的复杂度为 O(n)O(n)O(n), 其余部分对于每个 iii 都是 O(1)O(1)O(1) 的, 因此总复杂度为 O(n)O(n)O(n).
